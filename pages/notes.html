Caveats

Bookmarklets are executed as a single line of code. For this reason, lines without semicolons and // inline comments will cause syntax errors.

It's not possible for a bookmarklet to run more code after changing the current page. This is because whenevever a navigation happens, the page's current scripts are halted.
